# Model Evaluation, Tuning & Performance Analysis

## Overview

This project was completed as part of the Artificial Intelligence & Machine Learning Internship.

The objective was to evaluate, compare, and optimize machine learning models using a Netflix dataset. Logistic Regression and Decision Tree Classifier were trained and evaluated using classification metrics. Hyperparameter tuning was performed using GridSearchCV to improve model performance.

## Dataset

Netflix Movies and TV Shows Dataset

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Workflow

1. Data Loading
2. Data Preprocessing
3. Train-Test Split (80/20)
4. Logistic Regression Model
5. Decision Tree Model
6. Model Evaluation
7. Confusion Matrix Visualization
8. Hyperparameter Tuning using GridSearchCV
9. Model Comparison
10. Performance Analysis

## Results
The models were evaluated using Accuracy, Precision, Recall, and F1-Score. Hyperparameter tuning improved the performance of the selected model and demonstrated the importance of model optimization in machine learning projects.
